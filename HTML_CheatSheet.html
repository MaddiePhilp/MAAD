HTML Basics

Hyper Text Markup Language HTML forms the basis of how webpages are displayed on the internet. It is used to dictate where elements will be displayed on the page. The current iteration is HTML 5. Tags in HTML are shown by greater and less than symbols, for example <head>. For most tags the code end point is designated with the same word proceeded by a slash, for example </head>.

Good Practice and HTML Document Structure

The first line in any code should tell the computer or a person what langauge the code is writen in. In HTML this looks like <!DOCTYPE html>. The first level of the code (<html> </html>) shows where the HTML starts and ends. Everything within these should be indented by one tab. The next section of the code contains the <head> information like the page <title>. In this example the title is "My Cheatsheet! >.<". All the information within the head should be indented one more level (so two levels total). The </head> command is on the same indentation as the first <head> command. The next section in the document is the <body>. This contains the main contents of the page. This should also be indented within the <body> and </body> commands. Everyone has personal preferences but proper code structure looks something like this:

<!DOCTYPE html>
<html>
  <head>
    <title> My Title </title>
  </head>
  <style>
    .examplestyle{
      color: black;
      font-family: helvitica;
      }
  </style>
  <body>
    <h1>My Heading</h1>
    <p>My text block about something really<strong>awesome</strong>.</p>
  </body>
</html>

CSS Styles

Styles are built on top of html code using css to change text color, background color, borders, justification, margins, provide padding or fonts. style is an attribute of other tags. For example <p style="color:red">This is red!</p> creates "This is red!". These attributes can also be combined to change the font, color and size at the same time. For example the code <p style = "text-align:center; color:blue; font-family:impact; font-size:xx-large>I'm big, blue and bold now!</p> would produce:

Text Size

There are four different units to define size in the browser:

  1. Ems (Em) Ems are scalable. 1 relates to the current font size for example, if the font size is 12pt then 1em is 12pt. This makes them scalable, usefule for mobile web browsers.
  2. Pixels (Px) Pixels are a fixed size. 1px is 1 pixel on a computer screen. This mean pixel perfect measurement but it can not scale up or down for mobile users.
  3. Points (Pt) Points relate to printed media. 1pt is 1/72 of an inch. Therefore, like pixels, points can not scale in size.
  4. Percent (%) Percent is like em units. Apart from that 100% is the current font size. So if the font were 12pt then 12pt would equal 100%. This makes it scalable for mobile.

The font-size property can be set by a word or number attribute. For example of unit and number use try here or here.

Styling with CSS

CSS can control a lot of the webpage at once and can be applied to more than just fonts. There are three ways of defining CSS:

  1. Inline: By using the style attribute in HTML elements. This is how we changed the fonts in the last section.
  2. Internal: Using <style> elements in the <head>. There is an example of this at the top of the page.
  3. External: Using a CSS document. Useful if building or want to chagne an entire website easily. Not explored here.

Inline CSS

The most common uses of inline CSS have previously been discussed. It is likely only to be used if making only one item different. If using a repeated style it is quicker and easier to define the elements of the CSS in the head. To change words within a paragraph use the <span> tag. See examples in inline tag section.

Internal

Internally defining the css for a section of text can be done in the head of the document. It can be used to define all pargraphs or headings for the document:

<!DOCTYPE html>
<html>
  <head>
    <title> Defining in Style! </title>
  </head>
  <style>
    body{background-color:blue}
    h1{color:red; font-size:x-large}
    p{font-family:Comic Sans MS; color:red}
  </style>
</html>
